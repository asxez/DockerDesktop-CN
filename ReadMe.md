## 使用方法
1. 关闭Docker Desktop
2. 在Docker安装目录（默认为`C:\Program Files\Docker\Docker\frontend\resources`）找到app.asar文件并将其备份，防止出现意外。
3. 将从本仓库下载的asar文件改名为app.asar后替换原文件
